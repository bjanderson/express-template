Unauthorized Page - HTTPS

Install the self-signed certificate in your browser in order to gain access.

  1. Install server/ssl/ca.crt in your certificate authorities.
  2. Install server/ssl/client.p12 in your personal certificates. When prompted for a password, do not enter anything and just click OK. There is no password.
  3. If the certificates are expired, you can regenerate them by running the generate-certs.sh script in the server/ssl directory of the project.
